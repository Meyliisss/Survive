# Survive
Joc unde tu ești blocat într-o lume misterioasă, trebuie să aduni resurse, să construiești adăposturi și să lupți pentru a rămâne în viață. Fiecare zi aduce noi provocări — foamea, monștrii și frigul nopții. Explorează pământul, confecționează unelte și descoperă secretele ascunse sub suprafață.
